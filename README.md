# Superstore Sales Analysis

This project contains a small analytics pipeline for a Superstore sales dataset.

## Project Files

- `01_cleaning.py` - Loads the raw Superstore CSV, cleans the data, converts dates, and saves a cleaned CSV.
- `02_analysis.py` - Loads the cleaned data and produces summary metrics, category/region sales, top customers, and a monthly sales trend. It also exports charts and a summary CSV.
- `02_load_to_sql.py` - (Optional) Contains code to load the cleaned CSV into a SQL database or external table.
- `superstore.csv.csv` - Raw dataset source file.
- `superstore_cleaned.csv` - Cleaned dataset output generated by `01_cleaning.py`.
- `sales_summary.csv` - Summary metrics exported by `02_analysis.py`.
- `sales_analysis.png` - Generated visualization chart.
- `SQL/analysis_queries.sql` - SQL queries for analyzing sales data once the data is loaded into SQL.

## Requirements

- Python 3.13
- `pandas`
- `matplotlib`
- `seaborn`
- `sqlalchemy` (if using SQL load script)
- `pyodbc` (only if connecting to SQL Server)
- `openpyxl` (optional, for Excel support)

### 2. Analyze the cleaned data
