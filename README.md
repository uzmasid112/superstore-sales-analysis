# Superstore Sales Analysis

This repository contains an end-to-end data workflow for Superstore sales data. It demonstrates how to clean raw sales data, run analytics, and prepare SQL queries for future database analysis.

## Repository Contents

- `01_cleaning.py` — loads the raw dataset, cleans the data, converts date columns, and saves the cleaned output.
- `02_analysis.py` — loads the cleaned data and computes summary metrics, category and region sales, top customers, and monthly revenue trends. It exports charts and summary files.
- `02_load_to_sql.py` — optional script for loading the cleaned CSV into a SQL database.
- `superstore.csv.csv` — raw dataset source file.
- `superstore_cleaned.csv` — cleaned dataset output generated by `01_cleaning.py`.
- `sales_summary.csv` — summary metrics exported by `02_analysis.py`.
- `sales_analysis.png` — generated visualization chart.
- `superstore_dashboard.pbix` — Power BI dashboard file for the sales report.
- `SQL/analysis_queries.sql` — SQL query examples for later database analysis.
- `CONTRIBUTING.md` — contribution guidelines for the repository.

## Requirements

- Python 3.13
- pandas
- matplotlib
- seaborn
- sqlalchemy (optional, for SQL loading)
- pyodbc (optional, for SQL Server connections)
- openpyxl (optional, for Excel support)
