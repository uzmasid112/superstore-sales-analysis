# Contributing

Thank you for your interest in contributing to this project! The repository is built around Python data cleaning and analytics for Superstore sales data.

## How to contribute

1. Fork the repository on GitHub.
2. Clone your fork locally.
3. Create a new branch for your changes:

```bash
git checkout -b feature-name
```

4. Make your changes and test them locally.
5. Commit your changes with a clear message:

```bash
git commit -m "Add feature or fix description"
```

6. Push your branch to your fork:

```bash
git push origin feature-name
```

7. Create a pull request on GitHub.

## Suggestions for contributions

- Improve the data cleaning steps in `01_cleaning.py`.
- Add new analysis or visualization scripts.
- Add SQL loading examples in `02_load_to_sql.py`.
- Improve documentation or add more examples.

## Notes

- Keep changes focused and well-tested.
- If you change dataset paths, update the README accordingly.
- Avoid committing large binary files or unnecessary output files.
